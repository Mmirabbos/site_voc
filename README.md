# Vocabulary Learning App

Современное приложение для изучения английского языка с использованием алгоритма интервального повторения (Spaced Repetition) как в Anki и ReWord.

## Особенности

- **Интервальное повторение (SM-2)**: Алгоритм автоматически планирует повторения слов на основе того, насколько хорошо вы их помните
- **Светлая и темная темы**: Переключение между светлой и темной темами
- **Сохранение прогресса**: Весь прогресс сохраняется в localStorage браузера
- **Статистика**: Отслеживание серии дней, количества выученных слов и ежедневного прогресса
- **Красивый дизайн**: Современный интерфейс с анимациями и градиентами

## Как добавить свои слова

1. Создайте CSV файл в папке `public/voc/`
2. Формат файла:
\`\`\`csv
word,translation,type,tag
kitchen,кухня,noun,дом
cook,готовить,verb,кухня
\`\`\`

### Поля CSV:
- **word**: Английское слово
- **translation**: Перевод на русский
- **type**: Тип слова (noun, verb, adjective, adverb и т.д.)
- **tag**: Категория/тема (дом, кухня, сад, улица, врач и т.д.)

## Система оценок

При повторении слова вы оцениваете, насколько хорошо его помните:

- **Снова** (10 минут): Полностью забыли слово
- **Трудно** (1 день): Вспомнили с большим трудом
- **Хорошо** (1-6 дней): Вспомнили с усилием
- **Легко** (интервал увеличивается): Легко вспомнили

## Статусы карточек

- **Новое**: Слово, которое вы еще не учили
- **Изучаю**: Слово в процессе изучения
- **Повтор**: Слово на повторении
- **Выучено**: Слово успешно повторено 5+ раз
